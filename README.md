# 钢铁前线

一个无依赖的本地网页坦克对战游戏。

当前版本为 2.0.0：新增独立主界面、前线军人开场报告、总部多段爆炸与点击后结算流程；战绩会从旧版自动迁移，并继续跨版本保存在同一浏览器中。

## 运行

双击 `start.bat` 即可，启动器会优先使用 Google Chrome 打开 `http://127.0.0.1:4173`；未安装 Chrome 时会使用系统默认浏览器。服务器仅监听本机地址，不需要安装 Python、Node 或任何第三方依赖。

关闭启动窗口即可停止服务器。

## 操作

- 鼠标滚轮、拖动或手指滑动：上下查看纵向战场
- `1`—`7`：选择轻型坦克 / 中型坦克 / 重型坦克 / 野战炮 / 火箭炮 / 武装直升机 / 步兵装甲车
- `A` / `S` / `D`：在左翼 / 中路 / 右翼部署
- `空格`：暂停或继续
- 点击右侧小地图：快速移动战场视角
- 从右侧“防御工事”栏拖动拒马或机枪堡垒到战场：只能建在己方地面部队附近
- 点击战场中的我方单位：接管该单位；使用 `W` / `A` / `S` / `D` 自由移动，按 `Esc` 交还自动指挥

战场顶部提供三种全军调度命令：

- **进攻**：所有单位继续向敌军总部推进
- **防守**：全军撤到中央河流两座桥梁后方建立防线
- **全境防守**：所有单位回撤到我方基地周围，优先保护总部

手动操纵不会关闭自动武器系统，炮塔仍会自行搜索并攻击目标。敌方炮弹会落在开火时锁定的位置，不会在空中自动追踪；及时改变方向可以真正闪避炮弹。

轻型坦克消耗 3 点指挥点，移动快；中型坦克消耗 5 点，火力、生命和速度均衡；重型坦克消耗 7 点，速度慢但生命和伤害更高；野战炮消耗 6 点，防护和机动较弱，可在远距离发射高伤害曲射炮弹。火箭炮消耗 15 点，射程 420，一次连续发射 8 枚高伤害火箭，但完整装填需要 7.5 秒。

武装直升机消耗 8 点指挥点，可直接飞越河流，使用机枪攻击地面单位、总部、机枪塔和敌方直升机。普通地面单位无法攻击直升机；重型坦克炮塔两侧配有两门小型防空机枪，可以自动拦截空中目标。

步兵装甲车消耗 6 点指挥点。抵达战线后会停车打开后舱门，释放两名机枪步兵；步兵会跟随推进、自动寻找目标并使用机枪协同作战。

每个总部拥有 2000 点生命并配备两门独立炮台，会自动搜索、锁定并攻击进入射程的敌方单位。指挥点每 0.9 秒恢复 1 点，最多储存 20 点。摧毁敌方总部即可获胜。

拒马消耗 4 CP、施工 3 秒，可以阻挡并拖慢敌方地面单位；机枪堡垒消耗 9 CP、施工 6 秒，完成后会自动锁定并攻击空中或地面目标。施工期间会显示进度、脚手架、焊接火花和扬尘，工事也能被敌军摧毁。

每个基地前方另有两座机枪塔，可自动对空和对地射击。机枪塔能够被敌军摧毁，直升机也会主动攻击这些防空设施。

双方基地旁各有一座补给仓库。仓库存在时每 5 秒额外提供 1 点指挥点；仓库可以被摧毁，所有单位进入基地附近后会优先攻击仓库。敌方仓库同样会为 AI 提供少量额外补给。

战场中央有一条无法直接穿越的河流，只有左右两座桥可以通行。左翼和右翼会前往对应桥梁，中路部队会根据桥面交通自动选择较空闲的一座桥；过桥后再返回原推进路线。

开始战斗前可以选择四张地图：经典桥战保留双桥争夺；单桥决战拥有更宽的河流与唯一中央桥梁；平原战移除河流，适合大规模正面推进；大地图拥有更长的纵向战线和位于中央的双桥。

开始战斗前可以选择新手级、简单级、普通级、困难级或地狱级。难度越高，敌军指挥点恢复和增援调度越快，但本版本已整体放慢 AI 出兵节奏。敌方 AI 会主动积攒指挥点，混合部署坦克、野战炮、火箭炮、直升机与步兵装甲车，并根据前线威胁选择兵种和推进路线。敌军也会在地面部队附近施工拒马和机枪堡垒，难度越高允许同时维持的工事越多。

主界面战绩榜会统计胜利、战败、总战局和胜率。战绩使用浏览器本地存储保存，关闭游戏页面或退出当前战局后仍会保留。

同队单位会主动计算间距并向空隙分散，桥面上则降低横向避让力度，避免为了分散而驶入河中。
